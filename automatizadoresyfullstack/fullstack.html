Full Stack

¿Quién es?

Es el desarrollador cuyas funciones principales son diseñar, desarrollar sitios web y aplicaciones al igual que realizar su arquitectura.

Tiene que escribir el código en distintos lenguajes de programación tales como html, php o javascript.

También se tiene que encargar de definir la interfaz grafica de la aplicación o sitio web integrando, los gráficos, el audio y el video.

Una de sus muchas tareas es mantener la arquitectura y funcionalidad de la aplicación o sitio web al igual que diseñar y desarrollar nuevas funciones, tiene que trabajar en el back-end de una web para realizar cualquier cambio de la configuración en la misma. Esta persona tiene que utilizar las estrategias tanto front-end como back-end.

Los dos roles del Full Stack

Front-End

Que es el responsable por las partes del sitio web, con las personas que interactúan. Tiene que tomar el diseño de un website de un cliente o un equipo de diseño y escribe el código fundamental para implementarlo en la web.

Se tienen que asegurar que todo el contenido necesario para el sitio web es claro, visible y se encuentra en el lugar correcto, también asegurándose de que los colores correctos están donde deben estar, especialmente con respecto al color del texto, color de fondo y encabezados. Asegurándose que todos los enlaces estén correctos, que los botones funcionen bien y que el sitio web sea adaptable y atractivo.

Back-End

Se encargan del código detrás de escenas que controlan como carga y se ejecuta un sitio web.

Se tiene que ocupar del lado del servidor, creando el código y los programas necesarios para impulsar el servidor de la pagina web, la base de datos y cualquier otra aplicación que contenga.

Herramientas y Habilidades

Herramientas

  • Frameworks
  • Git / Github
  • Gestor de proyectos
  • Editores de Texto
  • Lengujes de programación
  • Control de las versiones
  • Extensiones de navegadores

Las buenas prácticas y habilidades de un desarrollador full-stack consisten en:

  • Tener que ser competente en al menos HTML, CSS, JavaScript
  • Crear código limpio y eficiente que haga lo que se requiere de la manera más rápida posible.
  • Supervisar el proceso completo en el caso de paginas que no tienen un gran presupuesto.
